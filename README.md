 ## Laravel Telegram Bot Notificatios
 How to install:
 

 - Clone the repository
 - Run composer install
 - Create your Telegram bot and channel.
 - Setup your bot name in `config/telegram.php` file
 - Set up you Telegram credentials in `.env` file.
